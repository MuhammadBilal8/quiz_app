Code of Conduct
===============

This project follows the Contributor Covenant code of conduct. By participating, you agree to respect others and follow the guidelines below.

Guidelines

- Be respectful and considerate in all communications.
- Refrain from harassment, discriminatory language, or personal attacks.
- Be welcoming to newcomers and help them learn.

Enforcement

If you experience or witness unacceptable behavior, please open an issue or contact the repository owner with as much detail as possible. The maintainers will respond and take action as appropriate.

For the full Contributor Covenant text and details, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
